## 🚀 Quick Start

```bash
# Clone & Setup
git clone https://github.com/syntax-in-orbit/a-world-away.git
cd a-world-away

# Environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate    # Windows

# Install & Run
pip install -r requirements.txt
python app.py
```

**Access:** `http://localhost:5000`
